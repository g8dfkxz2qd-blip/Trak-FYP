Both project file/application and code are available through the repository as this is a mobile app built in Xcode.
Instructions for accessing project file/code and running project:
Requirements – a mac computer running macOS with Xcode installed from the app store. If Xcode ha not already been installed make sure all component for simulator and development are downloaded too. Select all platform options to download.
Running the application – open the project through the Xcode home screen make sure that the device selector is set to a recent iPhone model e.g. iPhone 15,16,17. To run the app select the play icon in the top left or use cmd + r shortcut on your keyboard. The app should open in the simulator window it may take some time to load. A physical device is not required. 
Project file code is available through the left-hand menu pane of Xcode, select the folder icon in the menu which should be the first. The project code files should appear below click the files to view them.
